   - 텍스트 선택 → ✨ 개선: 매력적이고 읽기 쉽게 개선
   - 텍스트 선택 → 🎭 톤 변경: 친근한/전문적/유머러스 톤 전환
   - 텍스트 선택 → 📝 확장: 더 자세하고 풍부하게 확장
   - 텍스트 선택 → 📋 요약: 핵심 내용만 간결하게 요약
   - 텍스트 선택 → 🌍 번역: 다양한 언어로 번역
4. **다운로드**: HTML, Markdown 형식으로 다운로드
5. **키보드 단축키**: Ctrl+B(굵게), Ctrl+I(기울임), Ctrl+S(저장)

9. **독창성**: 차별화된 관점이나 새로운 정보
